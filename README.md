# Netflix Clone

A simple **Netflix Clone** built using **HTML**, **CSS**, and **JavaScript**. This project mimics the interface and basic functionality of Netflix, allowing users to browse through different movies and TV shows in a visually appealing way. ## Features

- **Responsive Design**: The app adjusts seamlessly to different screen sizes (desktop, tablet, mobile).
- **Movie/TV Show List**: Displays a list of movies/TV shows with posters and basic information.
- **Video Previews**: Hovering over a movie poster shows a preview of the movie or TV show.
- **Search Functionality**: Basic search feature to find movies and TV shows.
- **Navigation**: Includes a simple navbar for easy access to different sections.

## Technologies Used

- **HTML**: Structure of the webpage.
- **CSS**: Styling the page to mimic Netflix's layout.
- **JavaScript**: Dynamic content like fetching data from the API and handling interactions.
